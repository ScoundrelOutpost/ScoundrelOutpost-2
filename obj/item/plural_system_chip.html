plural_system_chip
/obj/item/plural_system_chip

Vars

fronter_nameThe name of the system's current fronter, which will be displayed on the ID card in place of the registered name.
lockedWhether the chip is locked or unlocked. When unlocked, it can be used in-hand to edit/add/remove system member names.
system_member_namesList of system member names. When the chip is attached to an ID, using the ID in-hand will allow the user to switch between these names.
system_nameThe name of the plural system, which is displayed on the ID card in {braces} at all times.
system_name_lockedWhen TRUE, prevents the system name from being modified. Used for the roundstart chips given by the plurality quirk. Set to FALSE when a HoP ID is swiped on the chip.

Procs

update_system_nameChanges the system_name variable and automatically updates the name of the chip to include it.
