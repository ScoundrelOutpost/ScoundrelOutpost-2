/datum/grand_eventDescribes something which can happen when the grand ritual is completed, usually not confined to the ritual location. Mostly this will be triggering events of escalating danger based on how many times the ritual has already been completed.
/datum/grand_event/eventA grand event which just triggers a normal event. Pretty simple.
/datum/grand_event/event/ghostAn event which requires some number of ghosts in order to run.
/datum/grand_event/midroundA grand event which triggers a dynamic midround
